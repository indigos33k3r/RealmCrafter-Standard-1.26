PlayerIsGM



Usage: 

		PlayerIsGM(Actor)



Description:
The PlayerIsGM command returns 1 if the specified actor has a GM account, or 0 if not. 0 is also returned if the actor specified is an NPC rather than a human player.



Example: 

		Function Main()
			GM = PlayerIsGM(Actor())
			If (GM == 1)
				Output(Actor(), "You have a GM account.")
			Else
				Output(Actor(), "You have a normal player account.")
			EndIf
			Return()
		End Function
