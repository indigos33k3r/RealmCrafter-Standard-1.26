ItemRange



Usage: 

		ItemRange(Item)



Description:
The ItemRange command returns the weapon range of an item retrieved using ActorWeapon, and similar commands. This value is meaningless for items which are not ranged weapons.



Example: 

		Function Main()
			MyRange = ItemRange(ActorWeapon(Actor()))
			Output(Actor(), "Your weapon has a range of " $+ MyRange)
			Return()
		End Function
