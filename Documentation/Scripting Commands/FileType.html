FileType



Usage: 

		FileType(Filename)



Description:
The FileType command returns the 2 if a given file is actually a folder, 1 if the file is a file, or 0 if the file does not exist.



Example: 

		Function Main()
			Type = FileType("Test.txt")
			If (Type == 1)
				Output(Actor(), "Test.txt is a file.")
			ElseIf (Type == 2)
				Output(Actor(), "Test.txt is a folder.")
			Else
				Output(Actor(), "Test.txt does not exist.")
			EndIf
			Return()
		End Function
