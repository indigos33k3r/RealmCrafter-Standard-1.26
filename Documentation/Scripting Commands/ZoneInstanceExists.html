ZoneInstanceExists



Usage: 

		ZoneInstanceExists(ZoneName, Instance)



Description:
The ZoneInstanceExists command returns 1 if the specified zone instance exists, or 0 if it does not. The default instance (#0) always exists.



Example: 

		Function Main()
			; Warp actor to Caves instance #5
			Result = ZoneInstanceExists("Caves", 5)
			If (Result == 0)
				CreateZoneInstance("Caves", 5)
			EndIf
			Warp(Actor(), "Caves", "Start", 5)
			Return()
		End Function
