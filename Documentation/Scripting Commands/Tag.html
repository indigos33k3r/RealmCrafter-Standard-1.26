Tag



Usage: 

		Tag(Actor)



Description:
The Tag command returns an actor's tag. The tag is a special addition to a character's nametag which can only be set using the SetTag command.



Example: 

		Function Main()
			MyTag = Tag(Actor())
			Output(Actor(), "You are a member of the " $+ MyTag $+ " guild")
			Return()
		End Function
