CallDLL



Usage: 

		CallDLL(DLLName, FunctionName[, InBank, OutBank])



Description:
The CallDLL command runs a function from a DLL. In order to work the DLL must be present in the same folder as Server.exe, or installed in the system32 folder. The function called must also have been exported in the following way: 

		extern "C"
		{
			_declspec(dllexport) int _cdecl MyDLLFunction(const void* in, int InSize, void* out, int OutSize);
		}
The optional bank parameters allow banks created with CreateBank to be passed between the script and the DLL in order to transfer parameters and return values.



Example: 

		Function Main()
			CallDLL("MyDLL", "MyDLLFunction")
			Return()
		End Function
