HasItem



Usage: 

		HasItem(Actor, Name, Amount)



Description:
The HasItem command returns 1 if the actor has the specified amount of an item in its inventory, or 0 if not.



Example: 

		Function Main()
			If (HasItem(Actor(), "Razor Mace", 1))
				Output(Actor(), "You have a mace!")
			Else
				Output(Actor(), "You do not have a mace")
			EndIf
			Return()
		End Function
