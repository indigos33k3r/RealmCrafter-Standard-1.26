ActorZ



Usage: 

		ActorZ(Actor)



Description:
The ActorZ command returns the current X position of an actor as a floating point value.



Example: 

		Function Main()
			MyX = ActorX(Actor())
			MyY = ActorY(Actor())
			MyZ = ActorZ(Actor())
			Output(Actor(), "Your position: " $+ MyX $+ ", " $+ MyY $+ ", " $+ MyZ)
			Return()
		End Function
