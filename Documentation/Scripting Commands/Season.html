Season



Usage: 

		Season()



Description:
The Season command returns the current season. This may be useful for events which only happen at certain times of year. The value returned is a number between 1 and the number of seasons created in the World Editor.



Example: 

		Function Main()
			Output(Actor(), "Season: " $+ Season()) ; Displays the current game season
			Return()
		End Function
