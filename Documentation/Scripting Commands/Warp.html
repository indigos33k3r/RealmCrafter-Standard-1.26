Warp



Usage: 

		Warp(Actor, ZoneName, PortalName, [Instance])



Description:
The Warp command moves an actor to any zone. It works on both player characters and NPCs. The optional Instance parameter allows you to warp an actor to any extra instance of the zone created with CreateZoneInstance.



Example: 

		Function Main()
			; Send actor to the caves
			Warp(Actor(), "Caves", "Start")
			Return()
		End Function
