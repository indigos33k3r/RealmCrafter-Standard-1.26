PlayMusic



Usage: 

		PlayMusic(Actor, MusicID, Global)



Description:
The PlayMusic command plays a music file once.
The Actor parameter specifies the player or players who will hear the sound. The MusicID parameter is the ID number of the music file to use. This can be found in the Media tab of the main editor. The Global parameter should be 0 to play the music only to the specified actor, or 1 to play the music to all actors in the same zone as the specified actor.



Example: 

		Function Main()
			PlayMusic(Actor(), 11, 0)
			Return()
		End Function
