Day



Usage: 

		Day()



Description:
The Day command returns the current day of the year. This may be useful for events which only happen at certain times of year. The value returned is a number between 1 and the year length set in the World Editor.



Example: 

		Function Main()
			Output(Actor(), "Day: " $+ Day()) ; Displays the current game day
			Return()
		End Function
