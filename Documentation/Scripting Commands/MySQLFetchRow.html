MySQLFetchRow



Usage: 

		MySQLFetchRow(QueryResult)



Description:
Fetch the next row from a query result



Example: 

		Function Main()

			Result = MySQLQuery("SELECT * FROM `mytable`")
			If(MySQLNumRows(Result) == 0)
				Return()
			EndIf	
			Row = MySQLFetchRow(Result)
			
		Return()			
		End Function
