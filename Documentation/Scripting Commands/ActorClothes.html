ActorClothes



Usage: 

		ActorClothes(Actor)



Description:
The ActorClothes command returns the body texture which the actor is using. The returned value is a number between 1 and 5.



Example: 

		Function Main()
			MyBody = ActorClothes(Actor())
			Output(Actor(), "You are using body texture " $+ MyBody)
			Return()
		End Function
