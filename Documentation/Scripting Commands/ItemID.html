ItemID



Usage: 

		ItemID(Item)



Description:
The ItemID command returns the fixed ID of an item instance retrieved using ActorHat, and similar commands. This value is the one displayed in GE when viewing items.



Example: 

		Function Main()
			ID = ItemID(ActorHat(Actor()))
			Output(Actor(), "Your hat has an ID of " $+ ID $+ " in the Game Editor")
			Return()
		End Function
