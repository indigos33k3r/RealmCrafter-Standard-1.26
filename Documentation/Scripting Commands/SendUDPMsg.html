SendUDPMsg



Usage: 

		SendUDPMsg(Stream, DestinationIP, [DestinationPort])



Description:
The SendUDPMsg command transmits all data written to the UDP stream to a specified IP address and port. Data should be written using the standard stream commands (e.g. WriteByte). If no destination port is specified, the port used to create the stream will be used.
Note that IP addresses must be in integer format. Use CountHostIPs to get an integer IP from a host in string format.
