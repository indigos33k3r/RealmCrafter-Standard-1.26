ActorLevel



Usage: 

		ActorLevel(Actor)



Description:
The ActorLevel command returns the level of an actor. All actors start at level one and increase in levels as they gain experience. See GiveXP.



Example: 

		Function Main()
			Result = ActorLevel(Actor())
			Output(Actor(), "You are at level " $+ Result)
			Return()
		End Function
