PlayerIsBanned



Usage: 

		PlayerIsBanned(Actor)



Description:
The PlayerIsBanned command returns 1 if the specified actor has a banned account, or 0 if not. 0 is also returned if the actor specified is an NPC rather than a human player.



Example: 

		Function Main()
			Ban = PlayerIsBanned(Actor())
			If (Ban == 1)
				; No point sending a message, the player is banned!
			Else
				Output(Actor(), "You are not banned, of course.")
			EndIf
			Return()
		End Function
