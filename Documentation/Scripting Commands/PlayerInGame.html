PlayerInGame



Usage: 

		PlayerInGame(Actor)



Description:
The PlayerInGame command returns 1 if the specified player actor is currently logged into the game, and 0 if it is an NPC or logged off.



Example: 

		Function Main()
			Human = PlayerInGame(Actor())
			If (Human == 1)
				Output(Actor(), "You are an online human player.")
			Else
				; Don't bother sending a message, NPCs or offline humans can't read them anyway!
			EndIf
			Return()
		End Function
