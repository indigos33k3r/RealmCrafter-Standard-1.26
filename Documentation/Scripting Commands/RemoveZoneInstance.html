RemoveZoneInstance



Usage: 

		RemoveZoneInstance(ZoneName, Instance)



Description:
The RemoveZoneInstance command will remove an instance previously created with CreateZoneInstance. The default instance (#0) cannot be removed.



Example: 

		Function Main()
			If (ZoneInstanceExists("Caves", 1))
				RemoveZoneInstance("Caves", 1)
			EndIf
			Return()
		End Function
