ItemWeaponType



Usage: 

		ItemWeaponType(Item)



Description:
The ItemWeaponType command returns the type of weapon for an item retrieved using ActorWeapon, and similar commands. Possible return values are:

1 - One handed
2 - Two handed
3 - Ranged



Example: 

		Function Main()
			MyHat = ItemValue(ActorHat(Actor()))
			Output(Actor(), "Your hat is worth " $+ MyHat)
			Return()
		End Function
