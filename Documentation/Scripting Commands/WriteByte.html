WriteByte



Usage: 

		WriteByte(Handle, Value)



Description:
The WriteByte command writes a value to a file as a single byte, at the current file position. The file position is then advanced by one byte.
The Handle parameter is the file handle returned by ReadFile, WriteFile or OpenFile. Value is the number to write to the file, and should be between 0 and 255.



Example: 

		Function Main()
			F = WriteFile("Test.dat")
				WriteByte(F, 100)
			CloseFile(F)
			Return()
		End Function
