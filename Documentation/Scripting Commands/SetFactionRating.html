SetFactionRating



Usage: 

		SetFactionRating(Actor, FactionName, Value)



Description:
The SetFactionRating command sets an actor's rating with a particular faction. The value set should be between -100 and 100.



Example: 

		Function Main()
			SetFactionRating(Actor(), "Elves", 100)
			Output(Actor(), "Set your popularity with elves to maximum.")
			Return()
		End Function
