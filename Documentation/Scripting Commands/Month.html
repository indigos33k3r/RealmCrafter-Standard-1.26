Month



Usage: 

		Month()



Description:
The Month command returns the current month. This may be useful for events which only happen at certain times of year. The value returned is a number between 1 and the number of months created in the World Editor.



Example: 

		Function Main()
			Output(Actor(), "Month: " $+ Month()) ; Displays the current game month
			Return()
		End Function
