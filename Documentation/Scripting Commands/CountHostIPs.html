CountHostIPs



Usage: 

		CountHostIPs(HostName)



Description:
The CountHostIPs command will enumerate all available IP addresses for a specified host, and return how many are available. You can then use the HostIP command to retrieve each one in integer format. You will need to use this to obtain integer format IP addresses for use with SendUDPMsg.
The HostName parameter should be a string containing either a dotted IP address (e.g. "127.0.0.1") or a host name (e.g. "realmcrafter.com").
