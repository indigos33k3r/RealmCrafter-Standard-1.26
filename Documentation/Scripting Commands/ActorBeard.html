ActorBeard



Usage: 

		ActorBeard(Actor)



Description:
The ActorBeard command returns the beard mesh which the actor is using. The returned value is a number between 1 and 5.



Example: 

		Function Main()
			MyBeard = ActorBeard(Actor())
			Output(Actor(), "You are using beard " $+ MyBeard)
			Return()
		End Function
