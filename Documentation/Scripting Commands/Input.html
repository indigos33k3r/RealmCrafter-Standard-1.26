Input



Usage: 

		Input(Actor, Title, Prompt, Numeric)



Description:
The Input displays a window where the player may enter any text, which is then returned to the script command. The Actor parameter must be an online player actor. The Title parameter is the text displayed as the title of the window, and the Prompt parameter is a short piece of text displayed next to the text entry box.
The Numeric parameter allows input to be restricted. Specify 0 to allow any input, 1 for text only, 2 for integers only, or 3 for decimal numbers only.



Example: 

		Function Main()
			Player = Actor()
			Result = Input(Player, "Please enter a new name for yourself", ">", 1)
			SetName(Player, Result)
			Return()
		End Function
