WaitItem



Usage: 

		WaitItem(Actor, Item, Number)



Description:
The WaitItem command pauses the script until an actor has in their inventory a certain amount of an item. This is useful for creating quest scripts.
The Item parameter is the name of the item which the actor must get. The Number parameter is the amount which the actor needs before the script will continue.



Example: 

		Function Main()
			; Wait for player to get 5 mushrooms
			WaitItem(Actor(), "Odd Mushroom", 5)
			Return()
		End Function
