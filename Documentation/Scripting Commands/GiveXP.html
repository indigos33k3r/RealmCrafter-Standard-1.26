GiveXP



Usage: 

		GiveXP(Actor, Amount, [DoNotShare])



Description:
The GiveXP command gives experience points to actor. It works on both player characters and NPCs. The actor will increase in level if you give them enough points. The optional DoNotShare parameter, if set to 1, will prevent the XP being shared with members of the actor's party.



Example: 

		Function Main()
			; Reward for quest completion
			GiveXP(Actor(), 150)
			Return()
		End Function
