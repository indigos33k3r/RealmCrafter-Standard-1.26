ItemValue



Usage: 

		ItemValue(Item)



Description:
The ItemValue command returns the value (in base currency units) of an item retrieved using ActorHat, and similar commands.



Example: 

		Function Main()
			MyHat = ItemValue(ActorHat(Actor()))
			Output(Actor(), "Your hat is worth " $+ MyHat)
			Return()
		End Function
