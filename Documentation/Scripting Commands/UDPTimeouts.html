UDPTimeouts



Usage: 

		UDPTimeouts(Time)



Description:
The UDPTimeouts command allows you to control how long (in milliseconds) the RecvUDPMsg function will wait for a new message. By default, this is set to 0, meaning RecvUDPMsg will return immediately if there is no message to be received. It is strongly recommended that this is left alone unless you know what you are doing, as it can affect normal server performance even when the script has ended.
