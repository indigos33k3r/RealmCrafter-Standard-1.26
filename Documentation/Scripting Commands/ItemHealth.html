ItemHealth



Usage: 

		ItemHealth(Item)



Description:
The ItemHealth command returns the health (as a percentage) of an item retrieved using ActorHat, and similar commands.



Example: 

		Function Main()
			MyHat = ItemHealth(ActorHat(Actor()))
			Output(Actor(), "Your hat is " $+ (100 - MyHat) $+ "% damaged")
			Return()
		End Function
