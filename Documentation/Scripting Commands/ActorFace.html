ActorFace



Usage: 

		ActorFace(Actor)



Description:
The ActorFace command returns the face texture which the actor is using. The returned value is a number between 1 and 5.



Example: 

		Function Main()
			MyFace = ActorFace(Actor())
			Output(Actor(), "You are using face texture " $+ MyFace)
			Return()
		End Function
