MySQLGetVar



Usage: 

		MySQLGetVar(Row, field)



Description:
Get the field value of a row.



Example: 

		Function Main()

			Result = MySQLQuery("SELECT * FROM `mytable`")
			If(MySQLNumRows(Result) == 0)
				Return()
			EndIf

			Row = MySQLFetchRow(Result)
			
			fName = MySQLGetVar(Row, "firstname")

		Return()
		End Function
