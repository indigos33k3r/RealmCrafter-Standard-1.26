ZoneOutdoors



Usage: 

		ZoneOutdoors(ZoneName)



Description:
The ZoneOutdoors command returns 1 if the specified zone is flagged as outdoors, or 0 if it is indoors.



Example: 

		Function Main()
			Result = ZoneOutdoors("Caves")
			If (Result == 0)
				Output(Actor(), "Zone is indoors")
			Else
				Output(Actor(), "Zone is outdoors")
			EndIf
			Return()
		End Function
