CreateFloatingNumber



Usage: 

		CreateFloatingNumber(Actor, Amount, Red, Green, Blue)



Description:
The CreateFloatingNumber command creates a floating number appear briefly above a character's head, like those seen in combat with the default system. The Actor parameter is the character to show the number above, Amount is the number to display, and the Red, Green and Blue parameters allow a colour to be set for the number.



Example: 

		Function Main()
			Player = Actor()
			; Display a red '-10' above the player's head
			CreateFloatingNumber(Player, -10, 255, 0, 0)
			Return()
		End Function
