PlayersInZone



Usage: 

		PlayersInZone(ZoneName, Instance)



Description:
The PlayersInZone command returns the total number of human players present in the specified zone. If -1 is supplied as the instance ID, the number returned will be the total number of players in every existing instance.



Example: 

		Function Main()
			Player = Actor()
			Result = PlayersInZone(ActorZone(Player), ActorZoneInstance(Player))
			Output(Player, "There are " $+ Result $+ " other players in your zone")
			Return()
		End Function
