WaitSpeak



Usage: 

		WaitSpeak(ActorA, ActorB)



Description:
The WaitSpeak command pauses the script until ActorA speaks to ActorB, by right clicking on him. This is useful for creating quest scripts.



Example: 

		Function Main()
			; Wait for player to speak to an NPC
			WaitSpeak(Actor(), ContextActor())
			Return()
		End Function
