FullTrim



Usage: 

		FullTrim(String)



Description:
The FullTrim command strips all leading and trailing spaces, tabs, and other control characters from a string and returns the result.



Example: 

		Function Main()
			Result = FullTrim(Chr(15) $+ "   Hello")
			Output(Actor(), Result) ; Displays Hello
			Return()
		End Function
