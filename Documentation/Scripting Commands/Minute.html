Minute



Usage: 

		Minute()



Description:
The Minute command returns the current minutes of the 24-hour game clock. Combined with the Hour() command you can get the full time of day. This may be useful for events which only happen at certain times of day.



Example: 

		Function Main()
			Time = Hour() $+ ":" $+ Minute()
			Output(Actor(), Time) ; Displays the current game time
			Return()
		End Function
