SetAbilityLevel



Usage: 

		SetAbilityLevel(Actor, Ability, Level)



Description:
The SetAbilityLevel command changes the level an ability is at, if the actor knows it. The level appears in the ability book and can be used by the ability script to alter the effects of the ability.



Example: 

		Function Main()
			; Increase healing spell to level 2
			SetAbilityLevel(Actor(), "Heal Self", 2)
			Return()
		End Function
