SetMoney



Usage: 

		SetMoney(Actor, Amount)



Description:
The SetMoney command sets the total money level of an actor to the amount specified. The amount is in base units (the lowest tier).



Example: 

		Function Main()
			SetMoney(Actor(), 5)
			Output(Actor(), "You have a total of 5 copper pieces!")
			Return()
		End Function
