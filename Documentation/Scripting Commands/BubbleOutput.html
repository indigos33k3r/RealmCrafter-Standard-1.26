BubbleOutput



Usage: 

		BubbleOutput(Actor, Message[, Red, Green, Blue])



Description:
The BubbleOutput commands allows you to display a message to players in a speech bubble above a character's head. The optional Red, Green and Blue parameters allow you to specifiy a colour other than black for the message, and should have a value between 0 and 255.



Example: 

		Function Main()
			BubbleOutput(Actor(), "Hello world!") ; Displays Hello World! above the actor's head
			Return()
		End Function
