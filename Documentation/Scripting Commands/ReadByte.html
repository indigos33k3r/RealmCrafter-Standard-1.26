ReadByte



Usage: 

		ReadByte(Handle)



Description:
The ReadByte reads a byte from the current file position and returns the value. The file position is then advanced by one byte.
The Handle parameter is the file handle returned by ReadFile, WriteFile or OpenFile.



Example: 

		Function Main()
			F = ReadFile("Test.dat")
				FileData = ReadByte(F)
				Output(Actor(), "First byte value: " $+ FileData)
			CloseFile(F)
			Return()
		End Function
