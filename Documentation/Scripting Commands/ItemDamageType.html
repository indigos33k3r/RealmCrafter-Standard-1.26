ItemDamageType



Usage: 

		ItemDamageType(Item)



Description:
The ItemDamageType command returns the weapon damage type of an item retrieved using ActorWeapon, and similar commands. This value is meaningless for items which are not weapons.



Example: 

		Function Main()
			MyDamageType = ItemDamageType(ActorWeapon(Actor()))
			Output(Actor(), "Your weapon does " $+ MyDamageType $+ " damage")
			Return()
		End Function
