ActorHair



Usage: 

		ActorHair(Actor)



Description:
The ActorHair command returns the hair mesh which the actor is using. The returned value is a number between 1 and 5.



Example: 

		Function Main()
			MyHair = ActorHair(Actor())
			Output(Actor(), "You are using hair " $+ MyHair)
			Return()
		End Function
