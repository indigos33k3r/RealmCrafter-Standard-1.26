SelectGadgetItem gadget,index

Parameters

gadget - A container gadget handle
index - Integer index of the item to select

Description

Sets the specified item to be the 'current' item.

The exact behaviour depends on the actual kind of gadget, but this typically involves the gadget
moving the specified item into view.

This command may only be used with combobox, listbox and tabber gadgets.
